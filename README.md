Listor Library
=================

PHP >= 5.3.3

A library to make pagination easy, sort it and filter it.

Requirements
------------

* PHP >= 5.3.3
* "symfony/event-dispatcher": "~2.3",
* "symfony/options-resolver": "~2.6"

Installation
------------

Can be installed with Composer:

```
$ composer require ws/listor
```
